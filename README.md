# RooVille 🦘

An interactive coastal town builder set on the Australian shore — create your avatar and build houses, businesses, docks, boats, boathouses, and zoos!

**Single-player only by design** — everything runs locally in the browser with no accounts, chat, or online features. Safe and private for kids.

## Play

```bash
npm install
npm run dev
```

Open the URL shown in your terminal (usually http://localhost:5173).

## Features

### Avatar creator
- Skin tone, hair, outfit, hats, accessories
- Pet companions: puppy, kitten, joey, parrot
- Getting around: walk, bike, or surfboard

### Custom illustrated buildings
- Hand-drawn SVG art for every building type
- Houses, businesses, docks, boats, boathouses, zoos, and decor

### Day / night cycle
- Automatic cycle through day, sunset, night, and dawn
- Tap the time button to change manually

### Sound effects
- Ocean waves, seagulls, placement sounds, quest fanfare
- Toggle sound on/off in the header

### Mini quests (Adventures)
- 6 quests to guide building — zoo keeper, harbour master, and more
- Progress tracked automatically with celebration when complete

### Touch / tablet support
- Full tap-and-drag support for iPad and touch screens

## How to play

1. Create your avatar
2. Pick buildings from the left menu
3. Tap the map to place them
4. Drag your character and buildings around
5. Complete adventures on the right panel
6. Your town saves automatically — no internet needed

## Tech

- React + TypeScript + Vite
- Saves to localStorage only
- No backend, no multiplayer, no data leaves the device

Built with love for coastal adventures! 🇦🇺
